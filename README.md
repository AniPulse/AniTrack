## üéØ About

**AniTrack** is a modern, responsive web application designed to help anime enthusiasts track their favorite seasonal anime releases. Built with Vue.js and inspired by AniList's sleek design, AniTrack provides a seamless experience for discovering and monitoring currently airing anime series.

### üéÆ Key Highlights

- üé® **AniList-Inspired Design**: Clean, dark theme with modern typography
- üì± **Fully Responsive**: Perfect experience across all devices
- ‚ö° **Real-time Data**: Powered by Jikan REST API for accurate anime information
- üîÑ **Live Countdowns**: Track upcoming episodes with real-time timers
- üîç **Smart Search**: Quickly find your favorite anime
- üåü **Seasonal Tracking**: Automatically detects and displays current season
